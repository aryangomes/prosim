# App Prosim

Converta números **decimais** em números **binários** ou em **hexadecimais**, ou ainda converta um **binário** em **hexadecimal** e vice-versa

**Operações disponíveis:**
- *ADD*
- *AND*
- *MOVE*
- *OR*
- *XOR*
- *COMPARE* 
- *INC (Incremento)*
- *NEG (Negação)*
- *SUB (Subtração)*
